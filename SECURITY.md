# Security Policy

## Supported Versions

Currently, only the latest `main` branch and the latest published NPM package version are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security issue in this project, please do not disclose it publicly. 
Instead, please report the vulnerability directly via [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) on the repository page.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress in fixing the issue.
